# 基于能量惩罚项求解VMC第一激发态 完整公式推导
## 核心思想
基态是能量泛函最小值，**第一激发态是正交于基态的最小能量态**。
在VMC中引入**能量惩罚+正交约束惩罚**，构造带惩罚的变分泛函，把激发态求解转化为带约束的极小化问题：
$$
\min_\theta \quad \mathcal{L}(\theta) = \langle E \rangle_\theta + \lambda \cdot \big|\langle \Psi_0 | \Psi_\theta \rangle\big|^2
$$
- $\Psi_0$：已收敛的**基态变分波函数**（固定不动）
- $\Psi_\theta$：待优化的**激发态试探波函数**
- $\lambda$：惩罚系数（足够大，强制 $\Psi_\theta \perp \Psi_0$）

---

## 1. 基础定义
设：
- 基态：$|\Psi_0\rangle$，已预先VMC收敛，固定参数
- 激发态试探态：$|\Psi(\theta)\rangle$
- 哈密顿量 $H$，满足 $H|\Psi_n\rangle = E_n|\Psi_n\rangle$，$E_0<E_1<E_2\cdots$

正交重叠：
$$
O(\theta) = \langle \Psi_0 | \Psi(\theta) \rangle
$$
重叠模方惩罚项：
$$
\mathcal{P}(\theta) = \big|O(\theta)\big|^2 = O^*(\theta)\,O(\theta)
$$

带惩罚总能量泛函：
$$
\mathcal{L}(\theta) = \langle E(\theta)\rangle + \lambda \cdot \big|\langle \Psi_0|\Psi(\theta)\rangle\big|^2
$$
$$
\langle E(\theta)\rangle = \frac{\langle\Psi(\theta)|H|\Psi(\theta)\rangle}{\langle\Psi(\theta)|\Psi(\theta)\rangle}
$$

---

## 2. VMC采样形式（蒙特卡洛可计算）
波函数模方作为采样分布：$\sigma \sim |\Psi(\sigma;\theta)|^2$

### 2.1 局部能量（不变）
$$
E_{\text{loc}}(\sigma;\theta) = \sum_\eta H_{\sigma\eta}\,\frac{\Psi(\eta;\theta)}{\Psi(\sigma;\theta)}
$$
能量期望：
$$
\langle E \rangle = \mathbb{E}_\sigma\big[E_{\text{loc}}(\sigma)\big]
$$

### 2.2 重叠项的采样表达
$$
\langle\Psi_0|\Psi_\theta\rangle
= \sum_\sigma \Psi_0^*(\sigma)\,\Psi(\sigma;\theta)
$$
归一化下采样估计：
$$
O(\theta) \approx \mathbb{E}_\sigma\left[ \frac{\Psi_0^*(\sigma)}{\Psi^*(\sigma;\theta)} \right]
$$

> 推导：
> $$
> \mathbb{E}_\sigma\left[\frac{\Psi_0^*(\sigma)}{\Psi^*(\sigma)}\right]
> =\sum_\sigma \frac{\Psi_0^*(\sigma)}{\Psi^*(\sigma)} \cdot \frac{|\Psi(\sigma)|^2}{\sum_{\sigma'}|\Psi(\sigma')|^2}
> =\frac{\sum_\sigma \Psi_0^*(\sigma)\Psi(\sigma)}{\sum_\sigma|\Psi(\sigma)|^2}
> \approx \langle\Psi_0|\Psi_\theta\rangle
> $$

---

## 3 惩罚泛函对参数的梯度（核心公式）
$$
\nabla_\theta \mathcal{L}
= \nabla_\theta \langle E \rangle
+ \lambda \,\nabla_\theta \big| \langle\Psi_0|\Psi_\theta\rangle \big|^2
$$

### 3.1 标准能量梯度（同基态VMC）
$$
\nabla_\theta \langle E\rangle
= 2\,\mathbb{E}_\sigma\Big[ \big(E_{\text{loc}}-\langle E\rangle\big)\,\nabla_\theta\log\Psi^*(\sigma) \Big]
$$

### 3.2 惩罚项梯度 解析推导
$$
\mathcal{P} = O^* O,\quad O=\langle\Psi_0|\Psi_\theta\rangle
$$
$$
\nabla_\theta \mathcal{P}
= O^*\nabla_\theta O + O\,\nabla_\theta O^*
$$

其中：
$$
\nabla_\theta O
= \nabla_\theta \langle\Psi_0|\Psi_\theta\rangle
= \langle\Psi_0|\nabla_\theta \Psi_\theta\rangle
$$

代入对数导数 $\nabla_\theta \Psi = \Psi\,\nabla_\theta\log\Psi$：
$$
\nabla_\theta O
= \mathbb{E}_\sigma\left[
\frac{\Psi_0^*(\sigma)}{\Psi^*(\sigma)} \,\nabla_\theta \Psi(\sigma)
\right]
= \mathbb{E}_\sigma\left[
\Psi_0^*(\sigma)\,\Psi(\sigma) \,\nabla_\theta\log\Psi(\sigma)
\right]
$$

最终**惩罚梯度**：
$$
\nabla_\theta \mathcal{P}
= O^* \,\mathbb{E}_\sigma\big[\Psi_0^*\Psi\,\nabla_\theta\log\Psi\big]
+ O \,\mathbb{E}_\sigma\big[\Psi_0\Psi^*\,\nabla_\theta\log\Psi^*\big]
$$

### 3.3 总激发态梯度（可直接代码用）
$$
\boxed{
\nabla_\theta \mathcal{L}
= 2\mathbb{E}\big[(E_{\text{loc}}-\langle E\rangle)\nabla_\theta\log\Psi^*\big]
+ \lambda \left(
O^* \mathbb{E}\big[\Psi_0^*\Psi\,\nabla_\theta\log\Psi\big]
+ O \mathbb{E}\big[\Psi_0\Psi^*\,\nabla_\theta\log\Psi^*\big]
\right)
}
$$

---

## 4 结合自然梯度/QGT的激发态更新
和基态一样用SR自然梯度：
$$
\theta_{k+1} = \theta_k - \alpha \cdot S_{\text{reg}}^{-1}\, \nabla_\theta \mathcal{L}
$$
- $S_{\text{reg}} = S + \lambda_{\text{diag}}I$ 仍是原QGT正则化
- 仅把**能量梯度替换为带惩罚的总梯度**即可

---

## 5 简化实用形式（工程常用）
实际代码中直接用离散采样平均代替期望：
1. 预存基态波函数 $\Psi_0(\sigma)$
2. 每步采样 $\{\sigma_i\}$
3. 计算：
   - $E_{\text{loc},i}$
   - $\log\Psi(\sigma_i),\nabla_\theta\log\Psi(\sigma_i)$
   - 重叠均值 $O = \frac1N\sum_i \Psi_0^*(\sigma_i)/\Psi^*(\sigma_i)$
4. 代入上面总梯度公式，再走QGT自然梯度更新

---

## 6 关键取值说明
1. $\lambda$ 惩罚系数：取 **5~20** 之间，越大正交约束越强，太大会训练震荡；
2. 基态 $\Psi_0$ **全程固定不优化**，只优化激发态网络；
3. 若求更高激发态：依次加入与前0,1,2…态的正交惩罚：
$$
\mathcal{L}(\theta) = \langle E\rangle + \sum_{n=0}^{k-1}\lambda_n \big|\langle\Psi_n|\Psi_\theta\rangle\big|^2
$$

# ===================== 4. 包装模型为 machine 函数 =====================
def create_machine(model: nnx.Module):
    """将 Flax NNX 模型包装为 NetKet 风格的 machine 函数"""
    graphdef, state = nnx.split(model)
    
    @jax.jit
    def machine(params, sigma):
        m = nnx.merge(graphdef, params)
        return m(sigma)
    
    return machine, graphdef, state

# ===================== 5. 纯 JAX 实现的 force-based 梯度计算 =====================
@partial(jax.jit, static_argnames=("machine",))
def compute_local_energies(machine, params, sigma):
    """
    计算局部能量 E_loc(σ) = Σ_η H(σ→η) ψ(η)/ψ(σ)
    
    这对应 NetKet 的 local_value_kernel
    """
    eta, H_eta = ha.get_conn_padded(sigma)
    logpsi_sigma = machine(params, sigma)
    logpsi_eta = machine(params, eta)
    logpsi_sigma = jnp.expand_dims(logpsi_sigma, -1)
    return jnp.sum(H_eta * jnp.exp(logpsi_eta - logpsi_sigma), axis=-1)


def statistics(x):
    """计算样本统计量"""
    mean = jnp.mean(x)
    var = jnp.var(x)
    return mean, jnp.sqrt(var / x.shape[0])


@partial(jax.jit, static_argnames=("machine",))
def forces_expect_hermitian(machine, params, sigma):
    """
    核心：复刻 NetKet 的 forces_expect_hermitian 函数
    
    使用 force-based 梯度计算：
    ∇⟨E⟩ = ⟨(E_loc - ⟨E⟩) ∇log ψ⟩
    
    关键：对于复数值网络，使用 holomorphic=True
    """
    # 1. 计算局部能量
    O_loc = compute_local_energies(machine, params, sigma)
    
    # 2. 统计能量均值
    O_mean, O_std = statistics(O_loc)
    
    # 3. 中心化局部能量
    O_centered = O_loc - O_mean
    
    # 4. 计算 ∇log ψ 对每个样本
    # 使用 jax.grad 计算复数梯度（holomorphic=True）
    def log_psi_single(p, s):
        return machine(p, s)
    
    def compute_grad_for_sample(s):
        return jax.grad(lambda p: log_psi_single(p, s), holomorphic=True)(params)
    
    grad_matrix = jax.vmap(compute_grad_for_sample)(sigma)
    
    # 5. 计算 force-based 梯度
    # grad = ⟨(E_loc - E_mean) ∇log ψ⟩ = (1/N) Σ (E_loc[i] - E_mean) ∇log ψ(σ[i])
    # grad_matrix 已经是 PyTree 结构，每个元素的形状是 (n_samples, ...)
    # 关键修复：O_centered 形状为 (n_samples,)，需要正确广播到梯度张量的每个维度
    # 使用 reshape 将 O_centered 变为 (n_samples, 1, 1, ..., 1) 以匹配梯度张量
    def weight_and_mean(grad_component):
        # grad_component 形状：(n_samples, d1, d2, ...)
        # O_centered 形状：(n_samples,)
        # 需要广播相乘后沿 axis=0 求平均
        weights = O_centered.reshape((O_centered.shape[0],) + (1,) * (grad_component.ndim - 1))
        return jnp.mean(weights * jnp.conj(grad_component), axis=0)
    
    grad = jax.tree_util.tree_map(weight_and_mean, grad_matrix)
    
    return O_mean, O_std, grad


#@partial(jax.jit, static_argnames=("machine",))
def compute_qgt(machine, params, sigma, diag_shift=0.1):
    """
    计算量子几何张量（QGT）/ F 矩阵
    
    QGT 定义：
    S_ij = ⟨∂_i log ψ* ∂_j log ψ⟩ - ⟨∂_i log ψ*⟩⟨∂_j log ψ⟩
    
    这就是 NetKet SR 的核心
    
    参数：
    - machine: 波函数机器
    - params: 网络参数
    - sigma: 样本 (n_samples, n_orbitals)
    - diag_shift: 对角线正则化参数 λ
    
    返回：
    - qgt_reg: 正则化后的 QGT 矩阵 (n_params, n_params)
    - unravel_fn: 用于将展平的向量恢复为 PyTree 结构的函数
    """
    n_samples = sigma.shape[0]
    
    # 步骤 1: 计算每个样本的 ∇log ψ
    def log_psi_single(p, s):
        return machine(p, s)
    
    def compute_grad_for_sample(s):
        return jax.grad(lambda p: log_psi_single(p, s), holomorphic=True)(params)
    
    # grad_matrix 是 PyTree，每个元素形状为 (n_samples, ...)
    grad_matrix = jax.vmap(compute_grad_for_sample)(sigma)
    
    # 步骤 2: 将 PyTree 展平为矩阵 (n_samples, n_params)
    grad_flat, unravel_fn = flatten_util.ravel_pytree(grad_matrix)
    grad_flat = grad_flat.reshape(n_samples, -1)
    
    # 步骤 3: 中心化（减去均值）
    # 这对应 QGT 定义中的第二项：- ⟨∂_i log ψ*⟩⟨∂_j log ψ⟩
    grad_mean = jnp.mean(grad_flat, axis=0, keepdims=True)  # (1, n_params)
    grad_centered = grad_flat - grad_mean  # (n_samples, n_params)
    
    # 步骤 4: 计算 QGT = (1/N) * Σ ∇log ψ* ∇log ψ^T
    # 注意：对于复数，需要使用共轭
    qgt = (1.0 / n_samples) * jnp.conj(grad_centered).T @ grad_centered
    
    # 步骤 5: 添加正则化
    qgt_reg = qgt + diag_shift * jnp.eye(qgt.shape[0])
    
    return qgt_reg, unravel_fn# ===================== 4. 包装模型为 machine 函数 =====================

@partial(jax.jit, static_argnames=("machine",))
def forces_expect_with_penalty(machine, params_es, params_gs, sigma_es, lambda_penalty):
    """
    修复 holomorphic 实数报错 + 完全匹配公式
    激发态 VMC 带正交惩罚梯度
    """
    # ===================== 1. 能量项 =====================
    E_loc = compute_local_energies(machine, params_es, sigma_es)
    E_mean, E_std = statistics(E_loc)
    E_centered = E_loc - E_mean

    # ===================== 2. 重叠 ⟨Ψ0|Ψes⟩ =====================
    psi_es = machine(params_es, sigma_es)
    psi_gs = machine(params_gs, sigma_es)
    psi_ratio = jnp.exp(psi_gs - psi_es)
    overlap = jnp.mean(psi_ratio)
    overlap_abs2 = jnp.abs(overlap) ** 2

    # ===================== 3. 总损失 =====================
    total_loss = E_mean.real + lambda_penalty * overlap_abs2

    # ===================== 4. 能量梯度 ∇⟨E⟩ =====================
    def log_psi(p, s):
        return machine(p, s)
    
    def grad_log_single(s):
        return jax.grad(lambda p: log_psi(p, s), holomorphic=True)(params_es)
    
    grad_log_psi = jax.vmap(grad_log_single)(sigma_es)

    def weight_grad(g):
        w = E_centered.reshape(-1, *((1,) * (g.ndim - 1)))
        return 2 * jnp.mean(w * jnp.conj(g), axis=0)
    
    grad_E = jax.tree_map(weight_grad, grad_log_psi)

    # ===================== 5. 惩罚梯度（修复 holomorphic 错误）=====================
    # 关键：penalty_fn 返回复数，再取实部，兼容 holomorphic=True
    def penalty_fn(p):
        psi_e = machine(p, sigma_es)
        ov = jnp.mean(jnp.exp(psi_gs - psi_e))
        # 用复数形式表达 |ov|² = ov * conj(ov)
        return lambda_penalty * ov * jnp.conj(ov)
    
    # 现在输出是复数，可以用 holomorphic=True
    grad_pen = jax.grad(penalty_fn, holomorphic=True)(params_es)

    # ===================== 6. 总梯度 =====================
    total_grad = jax.tree_map(lambda ge, gp: ge + gp, grad_E, grad_pen)

    return total_loss, E_mean, E_std, total_grad, overlap_abs2

for step in range(N_ITER):
    # 1. 采样

        machine, params, state=sampler_state,
        chain_length=20
    )
    samples = samples.reshape(-1, hi.size)
    ground_state_params = restore_state
    # 2. ✅ 正确调用带惩罚的梯度（关键修复）
