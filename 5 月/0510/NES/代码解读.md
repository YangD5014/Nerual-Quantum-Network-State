# NES-VMC 算法代码解读

## 一、算法概述

NES-VMC（Natural Excited State Variational Monte Carlo）是一种用于计算量子多体系统激发态能量的变分方法。其核心思想是将原系统前 $K$ 个激发态的求解问题等价转化为一个"扩展系统"的基态求解问题。

### 1.1 核心思想

传统的 VMC 只适用于基态计算，而 NES-VMC 通过构造特殊的行列式形式 Ansatz，可以同时优化多个态的变分参数，从而获得多个激发态能量。

### 1.2 与基态 VMC 的对比

| 特性 | 基态 VMC | NES-VMC |
|------|---------|---------|
| 目标 | 最小化 $\langle E \rangle$ | 最小化 $\mathrm{Tr}(E_L)$ |
| Ansatz | 单波函数 $\psi(x)$ | 行列式 $\Psi(\mathbf{x}) = \det[\psi_j(x^i)]$ |
| 损失函数 | 标量 $E_L$ | 矩阵 $E_L$ 的迹 |
| 输出 | 基态能量 | 前 $K$ 个激发态能量 |

## 二、核心代码实现

### 2.1 单态 Ansatz 定义

```python
class SingleStateAnsatz(nnx.Module):
    """单态 Ansatz：适配费米子系统的复数值 FFNN"""
    def __init__(self, n_spin_orbitals: int, hidden_dim: int = 16, *, rngs: nnx.Rngs):
        super().__init__()
        self.n_spin_orbitals = n_spin_orbitals
        self.linear1 = nnx.Linear(n_spin_orbitals, hidden_dim, rngs=rngs, param_dtype=complex)
        self.linear2 = nnx.Linear(hidden_dim, hidden_dim, rngs=rngs, param_dtype=complex)
        self.output = nnx.Linear(hidden_dim, 1, rngs=rngs, param_dtype=complex)

    def __call__(self, x: jax.Array) -> jax.Array:
        h = nnx.tanh(self.linear1(x))
        h = nnx.tanh(self.linear2(h))
        out = self.output(h)
        return jnp.squeeze(out)
```

**关键点**：
- 使用 Flax NNX 框架定义神经网络
- 所有参数类型为 `complex`（复数），因为费米子波函数需要相位信息
- 激活函数使用 `tanh`
- 输入是费米子占据数向量，输出是复数波函数振幅

### 2.2 NES 总 Ansatz（核心）

```python
class NESTotalAnsatz(nnx.Module):
    """NES-VMC 总 Ansatz：K 个单态 Ansatz 的行列式"""
    def __init__(self, n_spin_orbitals: int, n_states: int = 2, hidden_dim: int = 16, *, rngs: nnx.Rngs):
        super().__init__()
        self.n_states = n_states
        self.n_spin_orbitals = n_spin_orbitals

        key = jax.random.key(42)
        keys = jax.random.split(key, n_states)

        self.single_ansatz_list = [
            SingleStateAnsatz(n_spin_orbitals, hidden_dim, rngs=nnx.Rngs(keys[i]))
            for i in range(n_states)
        ]

    def __call__(self, x_batch: jax.Array) -> tuple[jax.Array, jax.Array]:
        if x_batch.shape[0] != self.n_states:
            raise ValueError(f"x_batch.shape[0] != {self.n_states}")

        K = self.n_states
        M = []
        for i in range(K):
            for j in range(K):
                psi_i_xj = self.single_ansatz_list[j](x_batch[i])
                M.append(psi_i_xj)

        M = jnp.stack(M, axis=0)
        M_matrix = M.reshape(K, K)
        psi_total = jnp.linalg.det(M_matrix)
        return psi_total, M_matrix
```

**核心思想**：
- 构建 $K \times K$ 矩阵 $\Psi_{ij} = \psi_j(x^i)$
- 返回值：行列式 $\Psi = \det(\Psi_{ij})$ 和矩阵 $M$
- 行列式形式保证了不同态之间的正交性约束

**数学意义**：
$$
\Psi(\mathbf{x}) = \det\begin{pmatrix}
\psi_1(x^1) & \psi_2(x^1) \\
\psi_1(x^2) & \psi_2(x^2)
\end{pmatrix}
$$

### 2.3 哈密顿量作用计算

```python
def Ham_psi(ha, model, x):
    """计算 Hψ(x)"""
    x_primes, mels = ha.get_conn(x)
    psi_values = jax.vmap(model)(x_primes)
    H_psi_x = jnp.sum(mels * psi_values)
    return H_psi_x
```

**计算步骤**：
1. `ha.get_conn(x)` 获取哈密顿量作用在 $x$ 上的所有连接态
2. `jax.vmap(model)` 批量计算所有连接态的波函数值
3. 加权求和得到 $H\psi(x)$

### 2.4 扩展哈密顿量作用

```python
def Ham_Psi(ha, total_ansatz, x):
    """计算扩展哈密顿量作用在总 Ansatz 上的矩阵"""
    k = total_ansatz.n_states
    H_psi_x_i = []
    for i in range(k):
        tmp = []
        for j in range(k):
            ele = Ham_psi(ha, model=total_ansatz.single_ansatz_list[j], x=x[i])
            tmp.append(ele)
        H_psi_x_i.append(tmp)

    HPsi = jnp.array(H_psi_x_i).reshape(k, k)
    return HPsi
```

**数学意义**：
$$
\hat{H}\Psi(\mathbf{x}) \triangleq \begin{pmatrix}
\hat{H}\psi_1(x^1) & \hat{H}\psi_2(x^1) \\
\hat{H}\psi_1(x^2) & \hat{H}\psi_2(x^2)
\end{pmatrix}
$$

### 2.5 局域能量矩阵计算（核心）

```python
def compute_local_energy_matrix(ha, total_ansatz, x_batch):
    """计算局域能量矩阵 E_L(x) = Ψ^{-1}(x) HΨ(x)"""
    psi_total, M_matrix = total_ansatz(x_batch)
    H_Psi = Ham_Psi(ha, total_ansatz, x_batch)

    E_L = jnp.linalg.solve(M_matrix, H_Psi.T).T
    return E_L, psi_total
```

**数学推导**：
- 局域能量矩阵定义为 $E_L(\mathbf{x}) \equiv \Psi^{-1}(\mathbf{x})\tilde{H}\Psi(\mathbf{x})$
- 使用 `jnp.linalg.solve` 求解线性系统 $\Psi \cdot E_L = \tilde{H}\Psi$
- 这对应论文中的公式 $\Psi^{-1}\hat{H}\Psi$

**物理意义**：
- 当 $K=1$ 时，退化为标准 VMC 的标量局部能量
- 当 $K>1$ 时，得到 $K \times K$ 矩阵，其本征值对应各激发态能量

### 2.6 NES 损失函数与梯度

```python
def compute_nes_loss_and_grad(ha, total_ansatz, params, sigma):
    """计算 NES-VMC 的损失函数和梯度"""
    E_L_list = []

    for i in range(sigma.shape[0]):
        E_L, _ = compute_local_energy_matrix_from_params(ha, total_ansatz, params, sigma[i])
        E_L_list.append(E_L)

    E_L_batch = jnp.stack(E_L_list)
    loss = jnp.trace(jnp.mean(E_L_batch, axis=0))

    grads = []
    for i in range(sigma.shape[0]):
        def loss_per_sample(p):
            E_L_s, _ = compute_local_energy_matrix_from_params(ha, total_ansatz, p, sigma[i])
            return jnp.trace(E_L_s)

        grad_i = jax.grad(loss_per_sample, holomorphic=True)(params)
        grads.append(grad_i)

    grad = jax.tree.map(
        lambda *x: jnp.mean(jnp.stack(x), axis=0),
        *grads
    )

    return loss, grad, E_L_batch
```

**损失函数**：
$$
\mathcal{L} = \mathrm{Tr}(\bar{E}_L) = \mathrm{Tr}\left(\mathbb{E}_{\mathbf{x} \sim \Psi^2}[E_L(\mathbf{x})]\right)
$$

**梯度公式**：
$$
\nabla_\theta \mathcal{L} = 2\mathbb{E}_{\mathbf{x} \sim \Psi^2}\left[(\bar{E}_L - \bar{E}_L)\nabla_\theta \log|\Psi(\mathbf{x})|\right]
$$

**实现细节**：
1. 遍历所有样本，计算每个样本的局域能量矩阵
2. 对损失函数求梯度，使用 `jax.grad(holomorphic=True)` 处理复数参数
3. 对所有样本的梯度取平均

### 2.7 本征值提取

```python
def extract_eigenvalues(E_L_matrices):
    """从局域能量矩阵提取本征值"""
    E_L_mean = jnp.mean(E_L_matrices, axis=0)
    eigenvalues, eigenvectors = jnp.linalg.eigh(E_L_mean)
    return eigenvalues, eigenvectors, E_L_mean
```

**物理意义**：
- 对平均局域能量矩阵 $\bar{E}_L$ 进行对角化
- 本征值即为前 $K$ 个激发态能量估计
- 本征向量给出对应的态组合系数

### 3.2 扩展希尔伯特空间采样

```python
def sample_extended_hilbert_space(sampler, machine, params, n_samples, K):
    """从扩展希尔伯特空间采样 K 组组态"""
    all_samples = []
    for _ in range(n_samples):
        sample, _ = sampler.sample(machine, params)
        all_samples.append(sample)

    # 重新组织为 K 组
    x_batches = []
    for i in range(K):
        batch_samples = all_samples[i::K]
        x_batches.append(jnp.stack(batch_samples))

    return jnp.stack(x_batches)
```

**关键点**：
- 扩展希尔伯特空间 $\mathcal{H}^{\otimes K}$ 由 $K$ 个原系统副本张量积构成
- 每次采样得到 $K$ 个组态 $\mathbf{x} = (x^1, \dots, x^K)$
- 这对应论文中的"一次性采样 K 组组态"

## 四、与标准 VMC 的对比

### 4.1 损失函数对比

**标准 VMC（基态）**：
$$
\mathcal{L}_{\text{VMC}} = \mathbb{E}_{x \sim |\psi|^2}[E_L(x)]
$$

**NES-VMC（激发态）**：
$$
\mathcal{L}_{\text{NES}} = \mathrm{Tr}\left(\mathbb{E}_{\mathbf{x} \sim |\Psi|^2}[E_L(\mathbf{x})]\right)
$$

### 4.2 梯度公式对比

**标准 VMC**：
$$
\nabla_\theta \mathcal{L} = 2\mathbb{E}_{x \sim |\psi|^2}[(E_L(x) - \bar{E})\nabla_\theta \log|\psi(x)|]
$$

**NES-VMC**：
$$
\nabla_\theta \mathcal{L} = 2\mathbb{E}_{\mathbf{x} \sim |\Psi|^2}[(\bar{E}_L - \bar{E})\nabla_\theta \log|\Psi(\mathbf{x})|]
$$

其中 $\bar{E}_L = \mathrm{Tr}(E_L)$

### 4.3 采样对比

**标准 VMC**：
- 采样分布：$|\psi(x)|^2$
- 样本形状：$(n\_samples, n\_orbitals)$

**NES-VMC**：
- 采样分布：$|\Psi(\mathbf{x})|^2$
- 样本形状：$(n\_samples, K, n\_orbitals)$

## 五、实现注意事项

### 5.1 复数梯度处理

```python
jax.grad(loss_per_sample, holomorphic=True)(params)
```

- 使用 `holomorphic=True` 告诉 JAX 函数是全纯的
- 自动处理复数梯度的计算规则
- 避免数值微分的高方差问题

### 5.2 行列式计算稳定性

```python
psi_total = jnp.linalg.det(M_matrix)
```

- 当 $K$ 较大时，行列式可能趋于零或无穷大
- 可以使用对数行列式或 LU 分解提高数值稳定性
- 注意检查行列式是否为零（会导致奇异矩阵）

### 5.3 线性系统求解

```python
E_L = jnp.linalg.solve(M_matrix, H_Psi.T).T
```

- 当 $\Psi$ 矩阵接近奇异时，求解可能不稳定
- 可以添加正则化项
- 使用 QR 分解提高数值稳定性

### 5.4 内存与计算成本

- 单态 Ansatz 参数数量：$O(n\_params)$
- NES 总 Ansatz 参数数量：$K \times O(n\_params)$
- 局域能量矩阵计算复杂度：$O(K^2 \times n\_connected)$
- 梯度计算复杂度：$O(n\_samples \times n\_params)$

## 八、总结

NES-VMC 通过引入行列式形式的总 Ansatz，将多个激发态的优化问题转化为单个扩展系统的基态优化问题。核心步骤包括：

1. **构造总 Ansatz**：使用 $K$ 个单态 Ansatz 的行列式
2. **计算局域能量矩阵**：$E_L = \Psi^{-1} H \Psi$
3. **优化损失函数**：最小化 $\mathrm{Tr}(E_L)$
4. **提取激发态能量**：对 $\bar{E}_L$ 对角化

这种方法的优势在于：
- 可以同时优化多个态
- 不需要显式正交化约束
- 可以复用标准 VMC 的采样和优化框架
