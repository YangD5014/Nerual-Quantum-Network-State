# H₂分子基态波函数分析

本项目使用量子计算技术计算H₂分子的精确基态波函数，并分析其在计算基下的表示形式。这是理解量子多体系统和开发量子蒙特卡洛算法的重要基础。

## 项目概述

H₂分子是最简单的双电子分子系统，其基态波函数包含了丰富的量子多体物理信息。通过精确对角化分子哈密顿量，我们可以获得：

1. **精确基态能量**
2. **波函数在计算基下的分解**
3. **电子关联效应的定量分析**
4. **轨道占据数分布**
5. **化学精确度验证**

## 主要结果

### 基态能量
- **计算值**: -1.857275 Hartree
- **实验参考**: -1.174 Hartree  
- **Hartree-Fock**: -1.133 Hartree
- **相关能**: -0.724 Hartree (-454.5 kcal/mol)

### 波函数结构
基态波函数主要由以下计算基构成：

| 排名 | 计算基 | 振幅 | 概率 | 物理意义 |
|------|--------|------|------|----------|
| 1 | \|0101⟩ | 0.9938 | 98.76% | Hartree-Fock参考态 |
| 2 | \|1010⟩ | -0.1115 | 1.24% | 双电子激发关联态 |
| 3-16 | 其他 | ~0.0000 | <0.01% | 高阶激发态 |

### 轨道占据数
- 轨道0: 0.9876 (主要占据)
- 轨道1: 0.0124 (少量占据)  
- 轨道2: 0.9876 (主要占据)
- 轨道3: 0.0124 (少量占据)

## 物理意义

### 1. Hartree-Fock近似
\|0101⟩ 态对应于Hartree-Fock近似下的基态，其中两个电子分别占据最低的两个自旋轨道。这个态占据了98.76%的权重，说明Hartree-Fock近似已经相当准确。

### 2. 电子关联效应
\|1010⟩ 态反映了双电子激发的关联效应，虽然只占1.24%，但对于获得化学精确度至关重要。这个态描述了电子之间的库仑排斥导致的关联行为。

### 3. 量子纠缠
概率熵为0.0966，表明系统存在适度的量子纠缠。主要的纠缠来自于两个重要基态的量子叠加。

## 计算方法

### 1. 分子哈密顿量构建
使用PySCF量子化学包构建H₂分子的电子结构哈密顿量：
```python
driver = PySCFDriver(
    atom="H 0 0 0; H 0 0 0.735",  # 键长: 0.735 Å
    basis="sto3g",                # 最小基组
    charge=0, spin=0,
    unit=DistanceUnit.ANGSTROM,
)
```

### 2. 二次量子化
将分子哈密顿量转换为二次量子化形式：
```python
second_q_op = hamiltonian.second_q_op()
```

### 3. Jordan-Wigner变换
使用Jordan-Wigner映射将费米子算符转换为量子比特算符：
```python
mapper = JordanWignerMapper()
qubit_op_ham = mapper.map(second_q_op)
```

### 4. 精确对角化
对量子比特哈密顿量进行精确对角化获得基态：
```python
ham_matrix = qubit_op_ham.to_matrix()
eigenvalues, eigenvectors = eigh(ham_matrix)
exact_energy = eigenvalues[0]
exact_state = eigenvectors[:, 0]
```

## 技术细节

### 量子比特映射
对于4量子比特系统，计算基\|q₃q₂q₁q₀⟩对应于：
- q₀, q₁: 第一个氢原子的自旋轨道
- q₂, q₃: 第二个氢原子的自旋轨道

### 化学精确度
