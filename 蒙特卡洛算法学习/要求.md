## Markdown公式显示经验总结

### 左右矢符号显示问题
**问题描述**：在量子力学文档中，左右矢符号（如 $|\psi\rangle$、$\langle\psi|$）在Markdown中显示不正确。

**解决方案**：
1. **正确的LaTeX语法**：
   - 右矢：使用 `|\psi\rangle` → 显示为 $|\psi\rangle$
   - 左矢：使用 `\langle\psi|` → 显示为 $\langle\psi|$
   - 内积：使用 `\langle\phi|\psi\rangle` → 显示为 $\langle\phi|\psi\rangle$
   - 外积：使用 `|\psi\rangle\langle\phi|` → 显示为 $|\psi\rangle\langle\phi|$

2. **转义字符处理**：
   - 确保在Markdown中正确转义反斜杠
   - 使用适当的LaTeX分隔符（`$...$` 或 `$$...$$`）

3. **验证方法**：
   - 在GitHub上预览文件验证显示效果
   - 使用本地Markdown查看器检查
   - 在Jupyter Notebook中测试公式的渲染

### 公式显示最佳实践
1. **一致性**：在整个文档系列中使用统一的公式语法
2. **可读性**：复杂的公式使用display模式（`$$...$$`）
3. **兼容性**：确保公式在不同平台（GitHub、Jupyter、本地查看器）都能正确显示
4. **测试**：创建测试文件验证公式显示效果

### GitHub特殊问题及解决方案

#### 问题描述
GitHub的Markdown渲染器对LaTeX公式支持有限，特别是量子力学中的左右矢符号（如 $|\psi\rangle$ 和 $\langle\psi|$）经常无法正确显示。

#### 完整解决方案
1. **自动化转换工具**：`convert_formulas.py` - 支持LaTeX与Unicode双向转换
2. **GitHub优化文档**：`01_quantum_mechanics_basics_github_unicode.md` - Unicode字符版本
3. **完整技术方案**：`GITHUB_FORMULA_SOLUTION.md` - 详细的技术文档
4. **测试对比文件**：`github_formula_test.md` - 各种方案的对比测试

#### 使用方法
```bash
# 预览转换效果
python convert_formulas.py preview

# 转换整个文件为Unicode版本（GitHub优化）
python convert_formulas.py convert -i input.md -o output_unicode.md --direction latex2unicode

# 反向转换回LaTeX版本
python convert_formulas.py convert -i input_unicode.md -o output_latex.md --direction unicode2latex
```
