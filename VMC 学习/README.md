# 变分神经网络量子态学习计划

## 欢迎来到变分神经网络量子态学习计划！

本学习计划旨在帮助您系统地学习变分神经网络量子态的理论基础、实现方法和应用实践。通过本计划，您将掌握从基础量子力学到高级神经网络量子态的完整知识体系，并能够独立实现变分蒙特卡洛方法来研究多体量子系统。

## 学习计划概览

本学习计划分为四个主要阶段：

### 阶段一：基础知识准备
- **量子力学基础**：了解量子力学的基本概念、波函数、薛定谔方程等
- **统计力学基础**：掌握统计力学的基本原理、系综理论、配分函数等
- **多体量子系统**：学习多体量子系统的描述方法、二次量子化等

### 阶段二：蒙特卡洛方法
- **蒙特卡洛基础**：理解随机数生成、蒙特卡洛积分、马尔可夫链等
- **变分蒙特卡洛方法**：掌握VMC的基本原理、试探波函数、局部能量等
- **采样算法**：学习各种采样方法，如Metropolis-Hastings、Gibbs采样等

### 阶段三：神经网络量子态
- **神经网络基础**：了解神经网络的基本结构、训练方法等
- **神经网络量子态**：学习如何用神经网络表示量子态
- **变分优化**：掌握神经网络量子态的优化方法

### 阶段四：实践项目
- **简单量子系统**：实现一维谐振子、氢原子等简单系统的VMC模拟
- **复杂量子系统**：研究多体量子系统，如Ising模型、海森堡模型等
- **实际应用**：探索神经网络量子态在量子化学、凝聚态物理中的应用
