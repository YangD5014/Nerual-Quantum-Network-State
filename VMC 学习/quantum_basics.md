# 量子力学基础

## 1. 波函数

量子力学中，波函数（Wave Function）是描述量子系统状态的数学函数，通常用希腊字母Ψ（Psi）表示。波函数包含了关于量子系统的所有信息。

### 波函数的物理意义

波函数的模平方 |Ψ|² 表示在特定位置找到粒子的概率密度。对于一维情况，粒子在位置 x 处的概率密度为：

P(x) = |Ψ(x)|²

对于三维情况，粒子在位置 r 处的概率密度为：

P(r) = |Ψ(r)|²

### 波函数的性质

1. **归一化条件**：整个空间中找到粒子的概率必须为1
   ∫|Ψ|²dτ = 1

2. **连续性**：波函数及其一阶导数必须是连续的

3. **单值性**：波函数必须是单值的

## 2. 薛定谔方程

薛定谔方程（Schrödinger Equation）是量子力学的基本方程，描述了量子系统如何随时间演化。

### 时间依赖的薛定谔方程

iℏ∂Ψ/∂t = ĤΨ

其中：
- i 是虚数单位
- ℏ 是约化普朗克常数
- Ĥ 是系统的哈密顿算符
- Ψ 是波函数

### 时间无关的薛定谔方程

对于定态（能量本征态），波函数可以表示为：

Ψ(r,t) = ψ(r)e^(-iEt/ℏ)

其中 ψ(r) 满足时间无关的薛定谔方程：

Ĥψ = Eψ

## 3. 量子态与测量

### 量子态

量子态是量子系统的完整描述，可以用波函数表示。在量子力学中，态叠加原理允许量子系统同时处于多个状态的叠加：

|Ψ⟩ = c₁|ψ₁⟩ + c₂|ψ₂⟩ + ... + cₙ|ψₙ⟩

其中 |ψᵢ⟩ 是基态，cᵢ 是复数系数，满足 |c₁|² + |c₂|² + ... + |cₙ|² = 1

### 测量

在量子力学中，测量会导致波函数坍缩到测量结果对应的本征态。测量得到某个本征值的概率为：

P(aₙ) = |⟨aₙ|Ψ⟩|²

其中 |aₙ⟩ 是对应于本征值 aₙ 的本征态。

### 不确定性原理

海森堡不确定性原理指出，某些物理量对（如位置和动量）不能同时被精确测量：

ΔxΔp ≥ ℏ/2

## 5. 量子系统的例子

### 一维无限深势阱

一维无限深势阱的势能函数为：

V(x) = 0, 0 ≤ x ≤ L
V(x) = ∞, 其他

对应的能量本征值和本征函数为：

Eₙ = n²π²ℏ²/(2mL²), n = 1, 2, 3, ...
ψₙ(x) = √(2/L)sin(nπx/L)

### 谐振子

一维谐振子的势能函数为：

V(x) = (1/2)mω²x²

对应的能量本征值和本征函数为：

Eₙ = (n + 1/2)ℏω, n = 0, 1, 2, ...
ψₙ(x) = (mω/πℏ)^(1/4) * (1/√(2ⁿn!)) * Hₙ(√(mω/ℏ)x) * e^(-mωx²/2ℏ)

其中 Hₙ 是厄米多项式。

## 6. 量子力学的数学形式

### 希尔伯特空间

量子态存在于希尔伯特空间中，这是一个完备的内积空间。波函数可以看作是希尔伯特空间中的矢量。

### 狄拉克符号

狄拉克符号是量子力学中常用的表示方法：

- |Ψ⟩：右矢（ket），表示量子态
- ⟨Ψ|：左矢（bra），表示右矢的对偶
- ⟨φ|Ψ⟩：内积，表示两个量子态的重叠

### 表象理论

量子力学可以在不同的表象中描述，常见的表象包括：

- 位置表象：波函数是位置的函数 Ψ(x)
- 动量表象：波函数是动量的函数 Φ(p)
- 能量表象：波函数用能量本征态展开

## 7. 量子力学与经典力学的对应

### 对应原理

在量子数很大的极限情况下，量子力学的结果应该与经典力学一致，这就是对应原理。

### 经典极限

当普朗克常数 ℏ 可以忽略不计时，量子力学过渡到经典力学。

## 8. 量子力学的应用

量子力学在现代科学和技术中有广泛的应用，包括：

- 原子物理和分子物理
- 凝聚态物理
- 粒子物理
- 量子计算和量子信息
- 量子化学

## 9. 量子力学与变分蒙特卡洛

在变分蒙特卡洛方法中，我们使用波函数来描述量子系统的状态，并通过优化波函数的参数来逼近基态能量。量子力学的基本原理为变分蒙特卡洛提供了理论基础。

### 变分原理

对于任意试探波函数 |Ψ_T⟩，系统的期望能量满足：

⟨Ĥ⟩ = ⟨Ψ_T|Ĥ|Ψ_T⟩/⟨Ψ_T|Ψ_T⟩ ≥ E₀

其中 E₀ 是系统的基态能量。这一原理是变分蒙特卡洛方法的基础。
