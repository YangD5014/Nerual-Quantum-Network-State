# 变分蒙特卡洛方法

## 1. 变分蒙特卡洛概述

变分蒙特卡洛（Variational Monte Carlo, VMC）是一种结合变分原理和蒙特卡洛采样方法，用于研究量子多体系统的强大工具。VMC允许我们使用参数化的试探波函数来近似真实量子系统的基态，并通过蒙特卡洛方法计算物理量的期望值。

### 变分原理

在量子力学中，变分原理指出：对于任意试探波函数 |ψ_T⟩，其能量期望值 ⟨H⟩ = ⟨ψ_T|H|ψ_T⟩/⟨ψ_T|ψ_T⟩ 总是大于或等于真实基态能量 E₀：

⟨H⟩ ≥ E₀

等号成立当且仅当 |ψ_T⟩ 是系统的真实基态。这为我们提供了一个优化波函数参数的准则：通过最小化能量期望值，我们可以得到尽可能接近真实基态的波函数。

### VMC的基本步骤

变分蒙特卡洛方法的基本步骤包括：

1. **选择试探波函数**：选择一个参数化的试探波函数 ψ_T(R; α)，其中 R 是所有粒子的坐标，α 是参数
2. **蒙特卡洛采样**：使用蒙特卡洛方法从概率分布 |ψ_T(R; α)|² 中采样
3. **计算局部能量**：对于每个采样点 R，计算局部能量 E_L(R; α) = (H ψ_T(R; α)) / ψ_T(R; α)
4. **计算能量期望值**：计算能量期望值 E(α) = ⟨E_L⟩
5. **优化参数**：使用优化算法（如随机梯度下降）调整参数 α 以最小化 E(α)

### VMC的优势

变分蒙特卡洛方法的主要优势包括：

1. **灵活性**：可以使用各种复杂的试探波函数，包括神经网络波函数
2. **无符号问题**：与某些量子蒙特卡洛方法不同，VMC没有符号问题
3. **可扩展性**：可以应用于大型量子系统
4. **并行性**：蒙特卡洛采样过程可以很容易地并行化

### VMC的局限性

变分蒙特卡洛方法的主要局限性包括：

1. **试探波函数依赖**：结果的准确性依赖于试探波函数的选择
2. **局部极小值**：优化过程可能陷入局部极小值
3. **统计误差**：蒙特卡洛采样引入统计误差
4. **计算成本**：对于复杂的波函数，计算局部能量可能很昂贵

## 2. 试探波函数

试探波函数的选择是变分蒙特卡洛方法的关键。好的试探波函数应该能够捕捉系统的关键物理特性，同时保持计算上的可行性。

### Jastrow波函数

Jastrow波函数是最常用的试探波函数之一，特别适用于描述相互作用粒子系统。其一般形式为：

ψ_T(R) = exp(∑ᵢⱼ u(rᵢⱼ))

其中 u(rᵢⱼ) 是Jastrow因子，通常选择为：

u(r) = (a/r)ᵐ 或 u(r) = a exp(-br)

Jastrow波函数能够描述粒子间的关联效应，但通常不足以描述复杂的量子多体现象。

### Slater-Jastrow波函数

Slater-Jastrow波函数结合了Slater行列式和Jastrow因子：

ψ_T(R) = D↑(R) D↓(R) exp(∑ᵢⱼ u(rᵢⱼ))

其中 D↑ 和 D↓ 分别是自旋向上和自旋向下的Slater行列式。这种波函数能够更好地描述费米子的交换反对称性。

### 反对称Jastrow波函数

反对称Jastrow波函数通过引入反对称的Jastrow因子来增强波函数的描述能力：

ψ_T(R) = exp(∑ᵢⱼ [u(rᵢⱼ) + (-1)^(σᵢ+σⱼ) v(rᵢⱼ)])

其中 σᵢ 是粒子 i 的自旋，v(r) 是反对称的Jastrow因子。

### 配对波函数

配对波函数特别适用于描述超导系统：

ψ_T(R) = det[f(rᵢⱼ)]

其中 f(rᵢⱼ) 是配对函数，通常选择为：

f(r) = exp(-r²/ξ²) 或 f(r) = 1/r

### 神经网络波函数

神经网络波函数是近年来发展起来的一种强大的试探波函数，其一般形式为：

ψ_T(R) = exp(NN(R; θ))

其中 NN(R; θ) 是一个神经网络，θ 是网络参数。神经网络波函数具有极强的表达能力，可以描述复杂的量子多体状态。

## 3. 局部能量

局部能量是变分蒙特卡洛方法中的核心概念，它允许我们将能量期望值的计算转化为对局部能量的平均。

### 局部能量的定义

对于试探波函数 ψ_T(R)，局部能量定义为：

E_L(R) = (H ψ_T(R)) / ψ_T(R)

其中 H 是系统的哈密顿量。能量期望值可以表示为：

⟨H⟩ = ∫ |ψ_T(R)|² E_L(R) dR / ∫ |ψ_T(R)|² dR = ⟨E_L⟩

即能量期望值等于局部能量的平均值。

### 局部能量的计算

计算局部能量需要计算哈密顿量作用于波函数的结果。对于一般的哈密顿量：

H = -∑ᵢ (ħ²/2mᵢ) ∇ᵢ² + ∑ᵢⱼ V(rᵢⱼ)

局部能量可以表示为：

E_L(R) = -∑ᵢ (ħ²/2mᵢ) [∇ᵢ² ψ_T(R)] / ψ_T(R) + ∑ᵢⱼ V(rᵢⱼ)

这要求我们能够计算波函数的拉普拉斯量。

### 拉普拉斯量的计算

对于一般形式的波函数 ψ_T(R) = exp(F(R))，我们有：

∇ᵢ ln ψ_T = ∇ᵢ F
∇ᵢ² ln ψ_T = ∇ᵢ² F
∇ᵢ² ψ_T / ψ_T = ∇ᵢ² ln ψ_T + (∇ᵢ ln ψ_T)²

因此，局部能量可以表示为：

E_L(R) = -∑ᵢ (ħ²/2mᵢ) [∇ᵢ² F + (∇ᵢ F)²] + ∑ᵢⱼ V(rᵢⱼ)

### 示例：一维谐振子

考虑一维谐振子，哈密顿量为：

H = -ħ²/2m d²/dx² + (1/2) mω²x²

选择试探波函数 ψ_T(x) = exp(-αx²)，则：

ln ψ_T = -αx²
d ln ψ_T/dx = -2αx
d² ln ψ_T/dx² = -2α

局部能量为：

E_L(x) = -ħ²/2m [-2α + (-2αx)²] + (1/2) mω²x²
       = ħ²α/m + (2ħ²α²/m - (1/2) mω²) x²

能量期望值为：

⟨E_L⟩ = ħ²α/m + (2ħ²α²/m - (1/2) mω²) ⟨x²⟩

对于概率分布 |ψ_T(x)|² = exp(-2αx²)/Z，有 ⟨x²⟩ = 1/(4α)，因此：

⟨E_L⟩ = ħ²α/m + (2ħ²α²/m - (1/2) mω²) / (4α)
       = ħ²α/m + ħ²α/(2m) - mω²/(8α)
       = (3ħ²α)/(2m) - mω²/(8α)

通过最小化 ⟨E_L⟩ 关于 α，我们可以得到最优参数。

## 4. 蒙特卡洛采样

在变分蒙特卡洛方法中，我们需要从概率分布 |ψ_T(R)|² 中采样。这通常使用马尔可夫链蒙特卡洛方法，如Metropolis-Hastings算法。

### Metropolis-Hastings算法

Metropolis-Hastings算法是一种从复杂概率分布中采样的通用方法。对于我们的情况，目标分布是 π(R) = |ψ_T(R)|²。

算法步骤如下：

1. 初始化：选择初始构型 R₀
2. 对于每个蒙特卡洛步骤 t = 1, 2, ..., N：
   a. 从提议分布 q(R'|Rₜ₋₁) 中采样候选构型 R'
   b. 计算接受概率：
      ```
      α(R'|Rₜ₋₁) = min[1, (|ψ_T(R')|² q(Rₜ₋₁|R')) / (|ψ_T(Rₜ₋₁)|² q(R'|Rₜ₋₁))]
      ```
   c. 从 U(0,1) 中采样 u
   d. 如果 u ≤ α(R'|Rₜ₋₁)，则接受 R' 并设置 Rₜ = R'；否则拒绝并设置 Rₜ = Rₜ₋₁

### 提议分布的选择

提议分布的选择对采样效率有重要影响。常用的提议分布包括：

1. **均匀分布**：
   ```
   R' = R + δ * U
   ```
   其中 U 是均匀分布在 [-1,1] 上的随机向量，δ 是步长参数。

2. **正态分布**：
   ```
   R' = R + δ * N(0,1)
   ```
   其中 N(0,1) 是标准正态分布，δ 是步长参数。

3. **物理启发式分布**：
   基于系统的物理特性设计的提议分布，例如在量子系统中，可以基于动能或势能来设计提议分布。

### 步长调整

步长参数 δ 的选择很重要：太小的步长会导致低接受率和高相关性，太大的步长也会导致低接受率。通常，我们调整步长使得接受率在 40%-60% 之间。

一种自动调整步长的方法是：

1. 初始化步长 δ₀
2. 运行一定数量的蒙特卡洛步骤，计算接受率
3. 如果接受率太低，减小步长；如果接受率太高，增加步长
4. 重复步骤2-3，直到达到目标接受率

### 预热期

在开始收集样本之前，我们需要运行一定数量的蒙特卡洛步骤，使系统达到平衡。这个阶段称为预热期（burn-in period）。

预热期的长度可以通过监控能量或其他物理量的演化来确定。当这些量达到稳定值时，可以认为系统已经达到平衡。

### 样本相关性

由于马尔可夫链的连续样本之间存在相关性，我们需要考虑这种相关性对统计误差的影响。有效样本数量可以估计为：

N_eff = N / (1 + 2∑ᵢ ρᵢ)

其中 N 是总样本数量，ρᵢ 是滞后 i 的自相关函数。

## 5. 能量优化

在变分蒙特卡洛方法中，我们需要优化波函数参数以最小化能量期望值。这通常使用基于梯度的优化方法。

### 能量梯度

能量关于参数 α 的梯度为：

∇ₐ⟨H⟩ = 2 [⟨E_L ∇ₐ ln ψ_T⟩ - ⟨E_L⟩⟨∇ₐ ln ψ_T⟩]

这个表达式可以通过"重权重"（reweighting）技术计算：

∇ₐ⟨H⟩ = 2 ⟨(E_L - ⟨E_L⟩) ∇ₐ ln ψ_T⟩

### 随机梯度下降

随机梯度下降（SGD）是一种常用的优化方法，其更新规则为：

αₜ₊₁ = αₜ - ηₜ ∇ₐ⟨H⟩

其中 ηₜ 是学习率，通常随时间递减。

### 自然梯度下降

自然梯度下降考虑了参数空间的几何结构，使用Fisher信息矩阵来调整梯度方向。其更新规则为：

αₜ₊₁ = αₜ - ηₜ F⁻¹ ∇ₐ⟨H⟩

其中 F 是Fisher信息矩阵，元素为：

Fᵢⱼ = ⟨(∇ᵢ ln ψ_T)(∇ⱼ ln ψ_T)⟩ - ⟨∇ᵢ ln ψ_T⟩⟨∇ⱼ ln ψ_T⟩

### 随机优化

在实际应用中，我们通常使用随机优化方法，即在每个优化步骤中只使用一部分样本来估计梯度。这可以大大提高计算效率。

常用的随机优化方法包括：

1. **随机梯度下降**（SGD）
2. **动量SGD**（Momentum SGD）
3. **AdaGrad**
4. **RMSProp**
5. **Adam**

### 约束优化

在某些情况下，我们需要在优化过程中施加约束，例如保持波函数的归一化。这可以通过拉格朗日乘子法或投影梯度法来实现。

## 6. 统计误差分析

在变分蒙特卡洛方法中，我们需要评估结果的统计误差，这通常通过自举法或块平均法来实现。

### 自举法

自举法是一种通过重采样来估计统计误差的方法。其步骤如下：

1. 从原始样本中有放回地随机抽取 N 个样本，形成一个新的样本集
2. 计算这个新样本集的能量期望值
3. 重复步骤1-2多次（例如1000次），得到能量期望值的分布
4. 计算这个分布的标准差，作为能量期望值的统计误差

### 块平均法

块平均法考虑了样本之间的相关性，其步骤如下：

1. 将样本序列分成若干块，每块包含 m 个连续样本
2. 计算每块的平均能量
3. 计算这些块平均能量的标准差，除以块数的平方根，得到能量期望值的统计误差

块大小 m 的选择很重要：太小的块不能消除相关性，太大的块会导致统计误差估计不准确。通常，我们选择 m 使得块之间的自相关可以忽略。

### 误差传播

对于依赖于波函数参数的物理量，我们需要考虑参数优化带来的误差。如果物理量 O 可以表示为 O(α)，则其误差可以通过误差传播公式估计：

σ²_O = (∂O/∂α)² σ²_α

其中 σ²_α 是参数 α 的方差。

## 7. 物理量的计算

除了能量之外，变分蒙特卡洛方法还可以计算其他物理量的期望值。

### 一般物理量的期望值

对于物理量算符 Ô，其期望值为：

⟨Ô⟩ = ⟨ψ_T|Ô|ψ_T⟩/⟨ψ_T|ψ_T⟩ = ∫ |ψ_T(R)|² O_L(R) dR / ∫ |ψ_T(R)|² dR = ⟨O_L⟩

其中 O_L(R) = (Ô ψ_T(R)) / ψ_T(R) 是局部物理量。

### 相关函数

相关函数是研究量子系统的重要工具。例如，密度-密度相关函数定义为：

C(r) = ⟨n(r) n(0)⟩ - ⟨n(r)⟩⟨n(0)⟩

在变分蒙特卡洛方法中，我们可以通过计算采样点上的局部相关函数，然后求平均来估计相关函数。

### 结构因子

结构因子是相关函数的傅里叶变换：

S(k) = 1/N ∑ᵢⱼ ⟨exp(ik·(rᵢ - rⱼ))⟩

结构因子提供了系统在动量空间中的信息，可以通过中子散射实验直接测量。

### 动量分布

动量分布描述了系统中粒子的动量状态：

n(k) = ⟨aₖ† aₖ⟩

在变分蒙特卡洛方法中，动量分布可以通过傅里叶变换波函数来计算。

## 8. 高级技巧

为了提高变分蒙特卡洛方法的效率和准确性，研究人员开发了多种高级技巧。

### 相关抽样

相关抽样是一种减少方差的技术，特别适用于计算能量差。其基本思想是使用相同的随机数序列来计算不同参数下的能量，从而减少统计波动。

### 力量偏差方法

力量偏差方法是一种优化波函数参数的替代方法，它最小化力量而不是能量。力量定义为：

F = ∫ |∇ψ_T|² dR

力量偏差方法通常比能量优化更稳定，但可能收敛到不同的解。

### 扩散蒙特卡洛

扩散蒙特卡洛（DMC）是一种投影方法，可以通过虚时间演化将试探波函数投影到真实基态。DMC通常与VMC结合使用，以获得更准确的结果。

### 绿色函数蒙特卡洛

绿色函数蒙特卡洛（GFMC）是一种基于格林函数的投影方法，特别适用于研究费米子系统。GFMC可以提供非常准确的结果，但计算成本较高。

### 路径积分蒙特卡洛

路径积分蒙特卡洛（PIMC）是一种基于路径积分表述的方法，特别适用于研究有限温度下的量子系统。PIMC可以处理玻色子系统，但对于费米子系统存在符号问题。

## 9. 神经网络量子态

神经网络量子态是近年来发展起来的一种强大的试探波函数，它使用神经网络来表示量子态。

### 神经网络波函数的基本形式

神经网络波函数的一般形式为：

ψ_T(R) = exp(NN(R; θ))

其中 NN(R; θ) 是一个神经网络，θ 是网络参数。这种形式确保波函数的相位是实数（可以推广到复数），并且可以表示任意复杂的量子态。

### 常见的神经网络结构

常见的神经网络量子态结构包括：

1. **前馈神经网络**（FNN）：最简单的神经网络结构，适用于小系统
2. **卷积神经网络**（CNN）：适用于具有平移对称性的系统
3. **循环神经网络**（RNN）：适用于一维系统
4. **图神经网络**（GNN）：适用于任意几何结构的系统
5. **Transformer**：适用于长程相互作用的系统

### 神经网络波函数的优势

神经网络波函数的主要优势包括：

1. **表达能力**：神经网络可以表示任意复杂的函数，理论上可以表示任意量子态
2. **可训练性**：可以使用深度学习中的优化技术来训练神经网络
3. **可扩展性**：神经网络可以处理大型系统
4. **灵活性**：可以针对特定系统设计专门的神经网络结构

### 神经网络波函数的挑战

神经网络波函数的主要挑战包括：

1. **计算成本**：计算局部能量可能很昂贵
2. **优化困难**：可能存在许多局部极小值
3. **解释性**：难以理解神经网络学到的物理特性
4. **泛化能力**：难以确保神经网络在未见过的构型上的表现

### 神经网络波函数的训练

训练神经网络波函数通常使用随机梯度下降或其变种。训练过程包括：

1. **初始化**：随机初始化神经网络参数
2. **采样**：使用Metropolis-Hastings算法从当前波函数中采样
3. **计算梯度**：计算能量关于参数的梯度
4. **更新参数**：使用优化算法更新参数
5. **重复**：重复步骤2-4直到收敛

## 10. 应用实例

变分蒙特卡洛方法在量子多体物理中有广泛的应用，以下是一些典型的应用实例。

### 量子自旋系统

量子自旋系统是研究量子磁性的理想平台。例如，对于一维海森堡模型：

H = J ∑ᵢ Sᵢ · Sᵢ₊₁

我们可以使用变分蒙特卡洛方法研究其基态性质，如磁化强度、相关函数等。

### 超流系统

超流系统是研究玻色-爱因斯坦凝聚的理想平台。例如，对于玻色-哈伯德模型：

H = -t ∑⟨ᵢⱼ⟩ (bᵢ† bⱼ + h.c.) + (U/2) ∑ᵢ nᵢ(nᵢ - 1)

我们可以使用变分蒙特卡洛方法研究其超流-莫特绝缘体相变。

### 超导系统
