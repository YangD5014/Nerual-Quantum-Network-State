# 蒙特卡洛基础

## 1. 蒙特卡洛方法概述

蒙特卡洛方法是一类基于随机采样的数值计算方法，由John von Neumann、Stanislaw Ulam和Nicholas Metropolis等人在20世纪40年代开发。这种方法利用随机数来模拟复杂系统的行为，特别适用于高维积分和复杂概率分布的采样。

### 蒙特卡洛方法的基本思想

蒙特卡洛方法的基本思想是通过随机采样来估计数学期望值。例如，要计算积分：

I = ∫ f(x) p(x) dx

其中 p(x) 是概率密度函数，我们可以通过从 p(x) 中采样 xᵢ，然后计算 f(xᵢ) 的平均值来估计 I：

I ≈ (1/N) ∑ᵢ f(xᵢ)

根据大数定律，当样本数量 N 趋近于无穷大时，这个估计值收敛于真实值。

### 蒙特卡洛方法的优势

蒙特卡洛方法的主要优势包括：

1. **维度无关性**：收敛速度与问题的维度无关，这使得它特别适用于高维问题
2. **灵活性**：可以处理复杂的积分区域和被积函数
3. **并行性**：采样过程可以很容易地并行化
4. **误差估计**：可以自然地提供统计误差估计

### 蒙特卡洛方法的局限性

蒙特卡洛方法的主要局限性包括：

1. **收敛速度慢**：收敛速度为 O(1/√N)，比确定性方法慢
2. **随机性**：结果具有随机性，需要多次运行以获得可靠结果
3. **方差问题**：对于某些问题，方差可能很大，需要更多样本

## 2. 随机数生成

随机数生成是蒙特卡洛方法的基础。在计算机中，我们通常使用伪随机数生成器来产生看似随机的数列。

### 伪随机数生成器

伪随机数生成器（PRNG）是通过确定性算法产生的数列，具有随机数的统计特性。常见的伪随机数生成器包括：

1. **线性同余生成器**（LCG）：
   ```
   X_{n+1} = (a X_n + c) mod m
   ```
   其中 a、c、m 是常数，X₀ 是种子。

2. **梅森旋转算法**（Mersenne Twister）：
   一种广泛使用的伪随机数生成器，具有长周期和良好的统计特性。

3. **Xorshift算法**：
   基于异或和位移操作的快速伪随机数生成器。

### 随机数质量检验

好的伪随机数生成器应该通过以下检验：

1. **均匀性检验**：生成的数应该在 [0,1] 区间内均匀分布
2. **独立性检验**：生成的数应该是统计独立的
3. **周期检验**：应该具有足够长的周期

### 从均匀分布到其他分布

从均匀分布 U(0,1) 生成其他分布的随机数的方法包括：

1. **逆变换法**：
   如果 F 是累积分布函数，F⁻¹ 是其逆函数，那么如果 U ~ U(0,1)，则 X = F⁻¹(U) 具有分布 F。

2. **接受-拒绝法**：
   要从目标分布 p(x) 中采样，选择一个易于采样的提议分布 q(x)，使得 p(x) ≤ M q(x) 对某个常数 M 成立。然后：
   - 从 q(x) 中采样 x
   - 从 U(0,1) 中采样 u
   - 如果 u ≤ p(x)/(M q(x))，则接受 x；否则拒绝并重复

3. **重要采样法**：
   从一个易于采样的分布 g(x) 中采样，然后通过权重 w(x) = p(x)/g(x) 进行校正。

## 3. 大数定律与中心极限定理

大数定律和中心极限定理是蒙特卡洛方法的数学基础。

### 大数定律

**弱大数定律**：如果 X₁, X₂, ..., Xₙ 是独立同分布的随机变量，期望值为 μ，则样本均值：

X̄ₙ = (1/n) ∑ᵢ Xᵢ

依概率收敛于 μ，即对于任意 ε > 0，

lim(n→∞) P(|X̄ₙ - μ| > ε) = 0

**强大数定律**：在相同条件下，X̄ₙ 几乎必然收敛于 μ，即

P(lim(n→∞) X̄ₙ = μ) = 1

### 中心极限定理

如果 X₁, X₂, ..., Xₙ 是独立同分布的随机变量，期望值为 μ，方差为 σ²，则当 n → ∞ 时，样本均值的分布趋近于正态分布：

√n (X̄ₙ - μ) → N(0, σ²)

这意味着对于大的 n，

X̄ₙ ≈ N(μ, σ²/n)

### 蒙特卡洛误差估计

根据中心极限定理，蒙特卡洛估计的标准误差为：

σ/√n

其中 σ 是单个样本的标准差，n 是样本数量。这表明：

1. 误差随样本数量的平方根减小
2. 要将误差减小一半，需要将样本数量增加四倍
3. 误差与问题的维度无关

## 4. 蒙特卡洛积分

蒙特卡洛积分是蒙特卡洛方法最基本的应用之一。

### 基本蒙特卡洛积分

要计算积分：

I = ∫ₐᵇ f(x) dx

我们可以将其重写为：

I = (b-a) ∫ₐᵇ f(x) [1/(b-a)] dx

其中 1/(b-a) 是 [a,b] 上的均匀分布的概率密度函数。然后，通过从 [a,b] 上的均匀分布中采样 xᵢ，计算：

I ≈ (b-a) (1/N) ∑ᵢ f(xᵢ)

### 高维积分

对于 d 维积分：

I = ∫ f(x) dx

其中 x ∈ ℝᵈ，我们可以从积分区域上的均匀分布中采样 xᵢ，然后计算：

I ≈ V (1/N) ∑ᵢ f(xᵢ)

其中 V 是积分区域的体积。

### 重要性采样积分

重要性采样是一种方差减少技术，特别适用于被积函数在某些区域变化剧烈的情况。选择一个重要性分布 g(x)，使得：

I = ∫ [f(x)/g(x)] g(x) dx = ∫ h(x) g(x) dx

其中 h(x) = f(x)/g(x)。然后，从 g(x) 中采样 xᵢ，计算：

I ≈ (1/N) ∑ᵢ h(xᵢ) = (1/N) ∑ᵢ f(xᵢ)/g(xᵢ)

理想情况下，g(x) 应该与 |f(x)| 成正比，这样 h(x) 的方差最小。

### 分层采样

分层采样是将积分区域划分为若干子区域，然后在每个子区域内进行采样。这种方法可以确保采样点在整个积分区域内均匀分布，从而减少方差。

## 5. 随机游走与马尔可夫链

随机游走和马尔可夫链是许多高级蒙特卡洛方法的基础。

### 随机游走

随机游走是一系列随机步骤的和。一维简单随机游走定义为：

Sₙ = X₁ + X₂ + ... + Xₙ

其中 Xᵢ 是独立同分布的随机变量，通常取值为 +1 或 -1，概率各为 1/2。

随机游走的性质：

1. 期望值：E[Sₙ] = 0
2. 方差：Var[Sₙ] = n
3. 扩散：均方位移 ⟨Sₙ²⟩ = n

### 马尔可夫链

马尔可夫链是一系列随机变量 X₀, X₁, X₂, ...，满足：

P(Xₙ₊₁ = x | X₀, X₁, ..., Xₙ) = P(Xₙ₊₁ = x | Xₙ)

即未来状态只依赖于当前状态，与过去状态无关。

#### 转移概率

马尔可夫链的转移概率定义为：

Pᵢⱼ = P(Xₙ₊₁ = j | Xₙ = i)

转移矩阵 P 的元素为 Pᵢⱼ。

#### 平稳分布

如果存在概率分布 π，使得：

πⱼ = ∑ᵢ πᵢ Pᵢⱼ

则 π 称为马尔可夫链的平稳分布。

#### 详细平衡条件

如果对于所有 i, j，有：

πᵢ Pᵢⱼ = πⱼ Pⱼᵢ

则称马尔可夫链满足详细平衡条件。满足详细平衡条件的马尔可夫链以 π 为平稳分布。

## 6. Metropolis-Hastings算法

Metropolis-Hastings算法是一种从复杂概率分布中采样的马尔可夫链蒙特卡洛方法。

### 算法步骤

1. 初始化：选择初始状态 x₀
2. 对于每个迭代步骤 t = 1, 2, ..., N：
   a. 从提议分布 q(x'|xₜ₋₁) 中采样候选状态 x'
   b. 计算接受概率：
      ```
      α(x'|xₜ₋₁) = min[1, (π(x') q(xₜ₋₁|x')) / (π(xₜ₋₁) q(x'|xₜ₋₁))]
      ```
   c. 从 U(0,1) 中采样 u
   d. 如果 u ≤ α(x'|xₜ₋₁)，则接受 x' 并设置 xₜ = x'；否则拒绝并设置 xₜ = xₜ₋₁

### Metropolis算法

Metropolis算法是Metropolis-Hastings算法的特例，其中提议分布是对称的，即 q(x'|x) = q(x|x')。在这种情况下，接受概率简化为：

α(x'|x) = min[1, π(x')/π(x)]

### 算法性质

1. **平稳分布**：如果算法满足某些条件（如不可约性和非周期性），则它以 π 为平稳分布
2. **详细平衡**：算法满足详细平衡条件
3. **收敛性**：经过足够多的步骤后，采样点将服从目标分布 π

## 7. Gibbs采样

Gibbs采样是一种特殊的马尔可夫链蒙特卡洛方法，适用于高维分布的采样。

### 算法步骤

对于 d 维分布 π(x₁, x₂, ..., x_d)：

1. 初始化：选择初始状态 x⁽⁰⁾ = (x₁⁽⁰⁾, x₂⁽⁰⁾, ..., x_d⁽⁰⁾)
2. 对于每个迭代步骤 t = 1, 2, ..., N：
   a. 从条件分布 π(x₁|x₂⁽ᵗ⁻¹⁾, x₃⁽ᵗ⁻¹⁾, ..., x_d⁽ᵗ⁻¹⁾) 中采样 x₁⁽ᵗ⁾
   b. 从条件分布 π(x₂|x₁⁽ᵗ⁾, x₃⁽ᵗ⁻¹⁾, ..., x_d⁽ᵗ⁻¹⁾) 中采样 x₂⁽ᵗ⁾
   c. ...
   d. 从条件分布 π(x_d|x₁⁽ᵗ⁾, x₂⁽ᵗ⁾, ..., x_{d-1}⁽ᵗ⁾) 中采样 x_d⁽ᵗ⁾

### 算法性质

1. **无条件接受**：Gibbs采样总是接受新状态，不需要计算接受概率
2. **条件分布**：算法依赖于能够从条件分布中采样
3. **收敛性**：与Metropolis-Hastings算法类似，Gibbs采样也以目标分布为平稳分布

## 8. 蒙特卡洛在统计物理中的应用

蒙特卡洛方法在统计物理中有广泛的应用，特别是在研究相变、临界现象和复杂系统方面。

### Ising模型

Ising模型是描述磁性系统的经典模型，其哈密顿量为：

H = -J ∑⟨ᵢⱼ⟩ sᵢ sⱼ - h ∑ᵢ sᵢ

其中 sᵢ = ±1 是格点 i 上的自旋，J 是交换耦合强度，h 是外磁场。

使用Metropolis算法模拟Ising模型的步骤：

1. 初始化：随机或有序地设置自旋构型
2. 对于每个蒙特卡洛步骤：
   a. 随机选择一个格点 i
   b. 计算翻转自旋 sᵢ 的能量变化 ΔE
   c. 如果 ΔE ≤ 0，接受翻转；否则以概率 e^(-βΔE) 接受翻转
3. 重复步骤2，直到系统达到平衡

### 连续变量系统

对于连续变量系统，如流体系统或粒子系统，可以使用类似的蒙特卡洛方法。例如，模拟经典流体系统的步骤：

1. 初始化：随机放置粒子
2. 对于每个蒙特卡洛步骤：
   a. 随机选择一个粒子
   b. 随机移动粒子一个小距离
   c. 计算能量变化 ΔE
   d. 如果 ΔE ≤ 0，接受移动；否则以概率 e^(-βΔE) 接受移动
3. 重复步骤2，直到系统达到平衡

## 9. 方差减少技术

方差减少技术是提高蒙特卡洛方法效率的重要手段。

### 重要性采样

重要性采样通过选择一个与被积函数成比例的采样分布来减少方差。理想情况下，采样分布应该与被积函数的绝对值成正比。

### 控制变量法

控制变量法利用一个与被积函数相关的已知函数来减少方差。例如，要计算 I = E[f(X)]，如果我们知道 E[g(X)] = μ，则可以定义：

Y = f(X) - c (g(X) - μ)

选择适当的 c 可以使 Y 的方差小于 f(X) 的方差。

### 分层采样

分层采样将采样空间划分为若干子区域，然后在每个子区域内进行采样。这种方法可以确保采样点在整个空间内均匀分布，从而减少方差。

### 对偶变量法

对偶变量法利用负相关的随机变量来减少方差。例如，如果 U ~ U(0,1)，则 1-U 也 ~ U(0,1)，且与 U 负相关。可以通过同时计算 f(U) 和 f(1-U) 来减少方差。

### 准蒙特卡洛方法

准蒙特卡洛方法使用低差异序列（如Halton序列、Sobol序列）代替伪随机数序列。这些序列在空间中更均匀地分布，可以加快收敛速度。

## 10. 蒙特卡洛与变分量子态

在变分量子态的研究中，蒙特卡洛方法扮演着重要角色。

### 变分蒙特卡洛

变分蒙特卡洛（VMC）是一种结合变分原理和蒙特卡洛采样方法，用于研究量子多体系统。其基本步骤包括：

1. 选择一个参数化的试探波函数 ψ_T(R; α)，其中 R 是所有粒子的坐标，α 是参数
2. 使用蒙特卡洛方法从概率分布 |ψ_T(R; α)|² 中采样
3. 计算局部能量 E_L(R; α) = (H ψ_T(R; α)) / ψ_T(R; α)
4. 计算能量期望值 E(α) = ⟨E_L⟩
5. 优化参数 α 以最小化 E(α)

### 量子蒙特卡洛

量子蒙特卡洛（QMC）是一类用于研究量子系统的蒙特卡洛方法，包括：

1. **变分蒙特卡洛**（VMC）：使用变分原理和蒙特卡洛采样
2. **扩散蒙特卡洛**（DMC）：基于虚时间演化的投影方法
3. **格林函数蒙特卡洛**（GFMC）：基于格林函数的投影方法
4. **路径积分蒙特卡洛**（PIMC）：基于路径积分表述的方法

### 蒙特卡洛在神经网络量子态中的应用

在神经网络量子态的研究中，蒙特卡洛方法用于：

1. **采样**：从神经网络波函数的概率分布中采样
2. **梯度计算**：计算能量关于神经网络参数的梯度
3. **优化**：使用随机梯度下降等方法优化神经网络参数
4. **误差估计**：估计能量和其他物理量的统计误差

## 11. 总结

蒙特卡洛方法是一类强大的数值计算方法，特别适用于高维积分和复杂概率分布的采样。在变分量子态的研究中，蒙特卡洛方法提供了计算期望值、优化参数和估计误差的工具。通过结合变分原理和蒙特卡洛采样，我们可以研究复杂的量子多体系统，并开发新的神经网络量子态方法。
