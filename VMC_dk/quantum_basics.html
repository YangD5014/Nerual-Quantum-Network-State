1. 量子力学简介

量子力学是描述微观粒子行为的物理理论，它与我们日常经验中的经典力学有着本质的区别。在量子力学中，粒子的行为由波函数描述，具有波粒二象性，并且测量过程会影响系统的状态。

量子力学的核心概念
  • 波粒二象性：微观粒子既表现出波的性质，又表现出粒子的性质
  • 测不准原理：无法同时精确测量粒子的位置和动量
  • 量子叠加：量子系统可以同时处于多个状态的叠加
  • 测量坍缩：测量会导致量子态坍缩到某个确定状态

量子力学的发展始于20世纪初，主要贡献者包括普朗克、爱因斯坦、玻尔、海森堡、薛定谔、狄拉克等。它成功地解释了经典力学无法解释的现象，如黑体辐射、光电效应、原子光谱等。

2. 波函数与概率解释

在量子力学中，系统的状态由波函数 $\Psi(x,t)$ 描述。波函数是一个复数函数，其物理意义由玻恩的概率解释给出：

$|\Psi(x,t)|^2$ 表示在时间 $t$ 在位置 $x$ 处找到粒子的概率密度

波函数需要满足归一化条件：

$\int_{-\infty}^{\infty} |\Psi(x,t)|^2 dx = 1$

这表示在整个空间中找到粒子的概率为1。波函数的相位（虽然不直接可观测）在量子干涉等现象中起着重要作用。

波函数的性质
  • 波函数是复数函数，可以表示为 $\Psi(x,t) = |\Psi(x,t)|e^{i\phi(x,t)}$
  • 波函数必须是单值、连续和有限的（除了在某些特殊点）
  • 波函数的平方模 $|\Psi(x,t)|^2$ 给出概率密度
  • 波函数的相位差会导致量子干涉现象

3. 薛定谔方程

薛定谔方程是量子力学的基本方程，描述了波函数随时间的演化。对于一维系统，时间依赖的薛定谔方程为：

$i\hbar\frac{\partial}{\partial t}\Psi(x,t) = \hat{H}\Psi(x,t)$

其中，$\hat{H}$ 是系统的哈密顿算符，通常形式为：

$\hat{H} = -\frac{\hbar^2}{2m}\frac{\partial^2}{\partial x^2} + V(x)$

这里，$m$ 是粒子质量，$V(x)$ 是势能函数，第一项代表动能，第二项代表势能。

定态薛定谔方程

当势能不显含时间时，可以使用分离变量法求解薛定谔方程。设波函数可以表示为：

$\Psi(x,t) = \psi(x)e^{-iEt/\hbar}$

代入时间依赖的薛定谔方程，得到定态薛定谔方程：

$-\frac{\hbar^2}{2m}\frac{d^2\psi(x)}{dx^2} + V(x)\psi(x) = E\psi(x)$

其中，$E$ 是系统的能量本征值，$\psi(x)$ 是对应的能量本征态。

示例：一维无限深势阱

考虑一个粒子在一维无限深势阱中运动，势能函数为：

$V(x) = \begin{cases} 0 & \text{if } 0 \leq x \leq L \\ \infty & \text{otherwise} \end{cases}$

定态薛定谔方程的解为：

$\psi_n(x) = \sqrt{\frac{2}{L}}\sin\left(\frac{n\pi x}{L}\right), \quad E_n = \frac{n^2\pi^2\hbar^2}{2mL^2}, \quad n=1,2,3,\dots$

4. 算符与期望值

在量子力学中，物理量由算符表示。算符是作用在波函数上的数学对象，对应于可观测的物理量。常见的算符包括：

  • 位置算符：$\hat{x} = x$（乘以x）
  • 动量算符：$\hat{p} = -i\hbar\frac{\partial}{\partial x}$
  • 动能算符：$\hat{T} = -\frac{\hbar^2}{2m}\frac{\partial^2}{\partial x^2}$
  • 哈密顿算符：$\hat{H} = \hat{T} + V(x)$

算符的本征方程为：

$\hat{A}\psi = a\psi$

其中，$a$ 是算符 $\hat{A}$ 的本征值，$\psi$ 是对应的本征态。

期望值

物理量 $A$ 在状态 $\Psi$ 中的期望值（平均值）为：

$\langle A \rangle = \int_{-\infty}^{\infty} \Psi^*(x,t)\hat{A}\Psi(x,t)dx$

对于归一化的波函数，期望值表示对该物理量进行多次测量的平均值。

算符的性质
  • 线性算符：$\hat{A}(c_1\psi_1 + c_2\psi_2) = c_1\hat{A}\psi_1 + c_2\hat{A}\psi_2$
  • 厄米算符：$\int \psi_1^*\hat{A}\psi_2 dx = \int (\hat{A}\psi_1)^*\psi_2 dx$，对应于可观测的物理量
  • 对易关系：$[\hat{A},\hat{B}] = \hat{A}\hat{B} - \hat{B}\hat{A}$，描述算符之间的相互关系

5. 量子态的叠加与测量

量子力学的一个基本原理是叠加原理，即如果 $\psi_1$ 和 $\psi_2$ 是系统的可能状态，那么它们的线性组合也是系统的可能状态：

$\Psi = c_1\psi_1 + c_2\psi_2$

其中，$c_1$ 和 $c_2$ 是复数系数。更一般地，量子态可以表示为多个本征态的叠加：

$\Psi = \sum_n c_n\psi_n$

其中，$\psi_n$ 是某个算符（如哈密顿算符）的本征态，$c_n$ 是对应的系数。

测量与坍缩

当对量子系统进行测量时，会发生以下过程：

  1. 波函数坍缩到某个本征态 $\psi_n$
  2. 测量结果为对应的本征值 $a_n$
  3. 测量得到 $a_n$ 的概率为 $|c_n|^2$

测量后，系统处于新的状态 $\psi_n$，除非系统处于本征态，否则测量会改变系统的状态。

示例：量子叠加态

考虑一个简单的量子系统，有两个可能的状态 $|0\rangle$ 和 $|1\rangle$。系统的叠加态可以表示为：

$|\Psi\rangle = \frac{1}{\sqrt{2}}|0\rangle + \frac{1}{\sqrt{2}}|1\rangle$

测量时，有50%的概率得到 $|0\rangle$，50%的概率得到 $|1\rangle$。测量后，系统坍缩到相应的状态。

6. 一维量子系统示例

一维谐振子

一维谐振子是量子力学中最重要的模型之一，其势能函数为：

$V(x) = \frac{1}{2}m\omega^2x^2$

其中，$m$ 是粒子质量，$\omega$ 是振荡频率。定态薛定谔方程的解为：

$\psi_n(x) = \left(\frac{m\omega}{\pi\hbar}\right)^{1/4}\frac{1}{\sqrt{2^nn!}}H_n\left(\sqrt{\frac{m\omega}{\hbar}}x\right)e^{-\frac{m\omega}{2\hbar}x^2}$
$E_n = \hbar\omega\left(n + \frac{1}{2}\right), \quad n=0,1,2,\dots$

其中，$H_n(x)$ 是厄米多项式。基态（$n=0$）波函数为高斯函数：

$\psi_0(x) = \left(\frac{m\omega}{\pi\hbar}\right)^{1/4}e^{-\frac{m\omega}{2\hbar}x^2}$

自由粒子

自由粒子的势能为零，$V(x) = 0$。定态薛定谔方程为：

$-\frac{\hbar^2}{2m}\frac{d^2\psi(x)}{dx^2} = E\psi(x)$

解为平面波：

$\psi_k(x) = \frac{1}{\sqrt{2\pi}}e^{ikx}, \quad E_k = \frac{\hbar^2k^2}{2m}$

其中，$k$ 是波数，与动量的关系为 $p = \hbar k$。自由粒子的波函数不是平方可积的，需要用波包来表示物理状态。

势垒隧穿

考虑一个粒子遇到高度为 $V_0$、宽度为 $a$ 的势垒：

$V(x) = \begin{cases} 0 & \text{if } x < 0 \text{ or } x > a \\ V_0 & \text{if } 0 \leq x \leq a \end{cases}$

当粒子能量 $E < V_0$ 时，经典力学中粒子无法越过势垒，但在量子力学中，粒子有一定概率"隧穿"通过势垒。隧穿概率为：

$T \approx e^{-2\kappa a}, \quad \kappa = \sqrt{\frac{2m(V_0-E)}{\hbar^2}}$

量子隧穿效应在许多物理现象和技术应用中起着重要作用，如放射性衰变、扫描隧道显微镜等。

7. 总结

量子力学是描述微观世界的基本理论，与我们熟悉的经典力学有着本质的区别。本节介绍了量子力学的基本概念，包括：

  • 波函数与概率解释：波函数描述量子系统的状态，其平方模给出概率密度
  • 薛定谔方程：描述波函数随时间的演化，是量子力学的基本方程
  • 算符与期望值：物理量由算符表示，期望值给出测量的平均值
  • 量子叠加与测量：量子态可以叠加，测量会导致波函数坍缩
  • 一维量子系统：介绍了一维谐振子、自由粒子和势垒隧穿等典型例子
量子力学的核心思想

量子力学的核心思想是微观粒子的行为由波函数描述，具有概率性和不确定性。测量过程会改变系统的状态，这是与经典力学的根本区别。量子力学成功地解释了微观世界的现象，并在现代科技中有着广泛的应用，如半导体、激光、核能、量子计算等。
